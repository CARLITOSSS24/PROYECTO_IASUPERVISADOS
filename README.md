# Sistema Inteligente de Transporte - Clasificación Supervisada

## Descripción del Proyecto
Este proyecto implementa un modelo de **Aprendizaje Supervisado** para predecir el estado operativo de las estaciones del sistema de transporte masivo. A diferencia de las fases anteriores, este modelo utiliza el algoritmo de **Árboles de Decisión (Decision Tree Classifier)** para aprender de datos históricos etiquetados y clasificar estaciones entre "Operación Normal" o "Retraso Crítico".
